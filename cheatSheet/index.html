HTML Elements

Elements

Tag Name Description
<body> The Document Body represents the content of an HTML document
<head> The Document Metadata (Header) contains machine-readable information (metadata) about the document
<title> The Document Title defines the document's title that is shown in a browser's title bar or a page's tab
<div> The Content Division generic container for flow content
<h1>-<h6> The Section Heading represent six levels of section headings. h1 is the highest section level and h6 is the lowest
<p> The Paragraph represents a paragraph
<span> The Content Span generic inline container.used to group elements for styling purposes
<em> The Emphasis marks text that has stress emphasis
<strong> The Strong Importance indicates that its contents have strong importance, seriousness, or urgency
<br> The line break a line break in text (for writing a poem or an address)
<a> The Anchor creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address
tables
<table> The Table represents tabular data
<thead> The Table Head defining the head of the columns of the table
<tbody> The Table Body encapsulates a set of table rows, indicating that they comprise the body of the table
<th> The Table Header defines a cell as header of a group of table cells
<tr> The Table Row defines a row of cells in a table
<td> The Table Data defines a cell of a table that contains data
<tfoot> The Table Foot defines a set of rows summarizing the columns of the table
